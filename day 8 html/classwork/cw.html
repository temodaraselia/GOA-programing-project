HTML

ჰიპერტექსტის მარკირების ენა (HTML) არის სტანდარტული მარკირების ენა[a] დოკუმენტებისთვის, რომლებიც შექმნილია ვებ ბრაუზერში საჩვენებლად. ის განსაზღვრავს ვებ კონტენტის შინაარსსა და სტრუქტურას. მას ხშირად ეხმარება ისეთი ტექნოლოგიები, როგორიცაა Cascading Style Sheets (CSS) და სკრიპტინგის ენები, როგორიცაა JavaScript. ვებ ბრაუზერები იღებენ HTML დოკუმენტებს ვებ სერვერიდან ან ლოკალური საცავიდან და ასახავენ დოკუმენტებს მულტიმედიურ ვებ გვერდებად. HTML აღწერს ვებ გვერდის სტრუქტურას სემანტიკურად და თავდაპირველად მოიცავდა მინიშნებებს მისი გარეგნობისთვის. HTML ელემენტები HTML გვერდების საშენი ბლოკებია. HTML კონსტრუქციებით, სურათები და სხვა ობიექტები, როგორიცაა ინტერაქტიული ფორმები, შეიძლება ჩაშენდეს რენდერირებულ გვერდზე. HTML უზრუნველყოფს სტრუქტურირებული დოკუმენტების შექმნის საშუალებას ტექსტის სტრუქტურული სემანტიკის აღნიშვნით, როგორიცაა სათაურები, აბზაცები, სიები, ბმულები, ციტატები და სხვა ელემენტები. HTML ელემენტები გამოყოფილია თეგებით, რომლებიც იწერება კუთხოვანი ფრჩხილების გამოყენებით. თეგები, როგორიცაა img და input, პირდაპირ შეაქვს შინაარსს გვერდზე. სხვა თეგები, როგორიცაა p და /p, გარს აკრავს და გვაწვდის ინფორმაციას დოკუმენტის ტექსტის შესახებ და შეიძლება შეიცავდეს ქვეელემენტების თეგებს. ბრაუზერები არ აჩვენებენ HTML ტეგებს, მაგრამ იყენებენ მათ გვერდის შინაარსის ინტერპრეტაციისთვის.
